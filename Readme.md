# 🏥 INDRA HOSPITAL WEBSITE

This is a beginner-friendly frontend hospital website project built using HTML, CSS, and basic responsive design concepts.


## Website Type

Healthcare / Hospital Website


## Technologies Used

* HTML5
* CSS3
* JavaScript (basic form validation, if included)
* GitHub (version control)
* Netlify (deployment)



## Pages Included

* Home (`index.html`)
* About (`about.html`)
* Services (`services.html`)
* Gallery (`gallery.html`)
* Appointment (`appointment.html`)
* Contact (`contact.html`)


## Features

* Responsive hospital website design for mobile, tablet, and desktop
* Fixed left sidebar navigation menu
* Clean hero section with hospital introduction
* Multi-page structure for better organization
* Appointment booking form
* Services section with departments and facilities
* Gallery section showcasing hospital infrastructure
* Contact page with hospital details
* SEO-friendly meta tags on all pages
* Fast-loading static website


## UI Design

* Primary Color: Blue (#0066cc)
* Secondary Color: Green (#00b894)
* Font Style: Arial / Google Fonts (Nunito, Inter)
* Layout: Left Sidebar + Main Content Area
* Style: Clean, modern, medical-themed UI


## How to Run

1. Download or clone the project
2. Open the folder in any code editor (VS Code recommended)
3. Open `index.html` in a browser
4. Use the sidebar to navigate between pages


## Netlify Deployment Steps

1. Create a GitHub repository
2. Upload all project files
3. Login to Netlify
4. Click **“Add new site” → “Import an existing project”**
5. Connect your GitHub repository
6. Click **Deploy Site**
